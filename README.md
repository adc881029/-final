## bulid process

```
pip install tk
```
## Introduction
我的程式從中央氣象局抓下台北市各區的氣象預報，並做呈現

## methods
我的程式的pseudocode
```
在圖形化介面添加下拉式選單、按鈕及輸出欄
if 如果按鈕被按下：
    執行main
    
def main()
    取得氣象預報資料
    if 資料的區域==下拉式選單中被選中的區域:
        把該區資料輸出到輸出欄
```
